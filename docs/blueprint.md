# **App Name**: Nexus Teams

## Core Features:

- Animated Landing Page: Create an engaging landing page with hero animations and a marquee showcasing live projects.
- Secure Authentication: Implement secure user authentication using email/password and Google OAuth.
- User Profiles: Allow users to create and manage profiles with skills, experience, language, and hackathon interests.
- Team Profiles: Enable teams to create profiles with project descriptions, open roles, and required skills.
- AI-Driven Team Formation: Use AI to suggest teams based on skills, passion, and availability, providing match scores and rationale, acting as a team-suggestion tool.
- Real-Time Chat with Moderation: Implement a real-time team chat with profanity filtering and moderation capabilities.
- Hackathon Listings: Display live and upcoming hackathons with event details and registration links.

## Style Guidelines:

- Dark Theme: Inspired by Batman, using a palette of black (#000000), deep gray (#333333), and red accents (#FF0000).
- Light Theme: A brighter theme for better contrast depending on user preferences.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look.
- Use simple and clear icons to represent skills, interests, and team roles.
- Mobile-first, responsive layout with clear sections and intuitive navigation.
- Subtle animations and microinteractions to enhance user engagement and provide feedback.
